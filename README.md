## Vue 3 + TypeScript + Vite + Pinia

- 基于 vue3.x 项目开发模板

## 项目集成

- `vue-router` 路由
- `pinia` 状态管理
- `vueuse` 常用工具库
- `eslint` 和 `prettier` 代码质量
- `git` 提交规范
- 集成`json-server` 本地模拟数据

## mock 数据

```bash
npm run mock  # 启动mock服务
# 或
yarn mock

```
