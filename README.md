## Vite + TypeScript + Vue3 + Pinia + Vueuse + Element Plus

- 基于 vue3.x 后台管理开发模板

## 项目集成

- `vue-router` 路由
- `pinia` 状态管理
- `vueuse` 常用工具库
- `element-plus` ui 库
- `eslint` 和 `prettier` 代码质量
- `git` 提交规范
- 集成`vite-plugin-mock` 本地模拟数据
